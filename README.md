# Asset Allocation (Streamlit)

App em Streamlit para comparar Ideal x Atual por blocos (macros):
- Renda Fixa Brasil (R$) por buckets (manual CSV)
- Renda Variável Brasil (R$) via yfinance
- Internacional (US$) via yfinance (tickers US / Charles Schwab)

O cenário "Neutro" fica como padrão, com opção de alternar para "Min" e "Max".
